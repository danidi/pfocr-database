caption: Biological pathways associated with the response to Ni(S-tcitr)2 in yeast.
  (a) Interaction maps of the genes that cause Ni(S-tcitr)2 sensitivity or resistance
  when deleted, generated with the functional annotation tool DAVID and visualized
  with the Enrichment Map plugin of Cytoscape. (b–d) same as (a) but applied to the
  indicated gene networks (‘chromatin/transcription’, ‘tubulin complex assembly’ and
  ‘Thr metabolism’ in panels b, c and d, respectively) and visualized with the Cytoscape
  (GeneMANIA plugin). Red and green nodes indicate genes that confer either Ni(S-tcitr)2
  resistance or sensitivity when deleted. Each node represents a functional category
  (e.g., a GO term or a KEGG pathway) and the width of the edges increases with the
  degree of gene overlap between the two connected categories; node size increases
  with the number of genes annotated to each functional category.
